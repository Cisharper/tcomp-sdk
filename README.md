# TComp SDK

- [Getting Started](#getting-started)

## Getting Started

### From npm/yarn (RECOMMENDED)

```
# yarn
yarn add @tensor-oss/tcomp-sdk
# npm
npm install @tensor-oss/tcomp-sdk
```

### From source

```sh
git clone https://github.com/tensor-hq/tcomp-sdk.git
cd tcomp-sdk/
yarn
# Build JS files
yarn tsc
```
